## 📋 What Is This?

This is a PowerBI dashboard made for exploring sales data from an online retailer that sells bicycles and bicycle parts. It lets you look at sales performance, customers, and products in clear, visual ways. You don’t need to know how to code or use databases. The dashboard is ready to use once you install it.

You can use this tool to:
- See trends in sales over time.
- Understand who your customers are.
- Track which products sell best.
- Identify patterns to improve your business decisions.

## ⚙️ Troubleshooting Tips

- **Dashboard doesn’t open:** Make sure Power BI Desktop is installed correctly.
- **Data looks empty:** Check if you opened the correct `.pbix` file.
- **Some visuals not loading:** Restart Power BI Desktop and try opening the file again.
- **Filters or buttons not working:** Power BI sometimes needs a quick refresh. Click **Home > Refresh** inside the app.

If problems continue, consult Microsoft’s Power BI support site or forums for help.
